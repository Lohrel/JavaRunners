Contexto:
Jogo no estilo runner (corrida infinita), onde o jogador corre constantemente a fim de alcançar uma pontuação (score) maior. O jogador encontra obstáculos pelo caminho (inimigos e objetos) que o atrapalham e encerram a partida em caso de colisão. O projeto também conta com um scoreboard (placar) que registra a pontuação máxima dos jogadores.
